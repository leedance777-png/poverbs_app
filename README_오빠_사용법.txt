잠언 APK 앱 프로젝트

1. Android Studio에서 이 폴더를 엽니다.
2. Build > Build APKs 를 누릅니다.
3. app/build/outputs/apk/debug/app-debug.apk 파일이 생성됩니다.

GitHub 무료 자동빌드
1. 이 폴더 내용을 GitHub 저장소에 업로드합니다.
2. Actions 탭 > Build Android APK > Run workflow 를 누릅니다.
3. 완료 후 Artifacts에서 proverbs-debug-apk를 다운로드합니다.

특징
- 서버비 0원
- 앱 내부 assets에 잠언 31장 JSON 포함
- 인터넷 없이 실행
- 카카오톡 브라우저가 아니라 진짜 APK 앱이라 안드로이드 네이티브 TTS 사용
- 절 터치: 그 절부터 계속 읽기
- 길게 터치: 즐겨찾기/노트
- 하단: 즐겨찾기/노트/설정
